- Next.js
- React
- TypeScript
- Tailwind CSS

## Cara Menjalankan Proyek

1. Install dependencies:

```bash
npm install
```

2. Jalankan server development:

```bash
npm run dev
```

3. Buka di browser:

```text
http://localhost:3000
```
